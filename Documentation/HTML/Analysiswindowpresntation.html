Ctrl_GraphWindow ››

Screen shot below shows a typical analysis window graphic drawn using default properties.



The analysis window has four main areas:


    • The graphic where traces are drawn
    • The channel list, on the left, where all channels available in the loaded data file are displayed
    • The legend, on the right, where all channel plotted as well as their values, units and statistics are shown.
    • The tool bar, on the top, where all principal command of the graphic window are available
